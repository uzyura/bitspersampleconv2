# WasapiBitmatchChecker #

WasapiBitmatchChecker is a bitmatch transfer check program for PC audio devices.

# System requirements #

  * Windows 7 x64 or later
  * 8GB or more memory
  * PC audio interface with digital output
  * PC audio interface with digital input
